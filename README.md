# MAPAP — Проектный офис по транспорту и логистике РК

Система управления задачами Администрации по логистике и транспорту Республики Казахстан.

🔗 **[Открыть демо →](https://oelunchik.github.io/mapap)**

---

## Возможности

- **Дашборд** — KPI-карточки, сводная таблица по сотрудникам, просроченные задачи
- **Задачи** — список 52 задач из PRD с фильтрами по сотруднику, направлению, статусу, приоритету
- **Канбан-доска** — 5 колонок (Новая → В работе → На проверке → Готово → Заблокировано), drag & drop
- **Международные треки** — 8 партнёров (IATA, IRU, DPWorld, Global Gateway и др.)
- **Отчёты** — сводная таблица с прогресс-барами, экспорт CSV
- **Авторизация** — роли руководитель / аналитик с разграничением прав

## Деплой (GitHub Pages)

Сайт деплоится автоматически из ветки `main`. Файл `index.html` в корне репозитория — полностью автономное приложение без зависимостей от сборщика.

## Полная версия (Docker)

Для production-деплоя с PostgreSQL + Redis + REST API:

```bash
git clone https://github.com/Oelunchik/mapap
cd mapap
cp .env.example .env
docker compose up -d
# → http://localhost
```
